Planned interfaces

Agent access for metabolic consult workflows.

lifelongpep plans to support assistants and automations through MCP, CLI, and API workflows for consult preparation, AI education, and doctor booking after launch.

Planned examples

lifelongpep.prepare_consult({ goal: "GLP-1 readiness", mode: "education" })
lifelongpep consult prepare --goal glp1-readiness --handoff doctor
POST /v1/consult-intents { "intent": "glp1-readiness" }

What agents should be able to do

Prepare

Collect goals, history, metabolic context, and readiness questions before a consult.

Educate

Use an AI layer for general education and next-step guidance, not diagnosis.

Book

Request doctor consultation options once provider and booking workflows are live.
